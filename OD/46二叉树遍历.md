# 二叉树遍历

**题⽬描述**

根据给定的⼆叉树结构描述字符串，输出该⼆叉树按照中序遍历结果字符串。中序遍历顺序为:左⼦树，根结点，右⼦树。

**输⼊描述**

- 由⼤⼩写字母、左右⼤括号、逗号组成的字符串:
- 1、字母代表⼀个节点值，左右括号内包含该节点的⼦节点。
- 2、左右⼦节点使⽤逗号分隔，逗号前为空则表⽰左⼦节点为空,没有逗号则表⽰右⼦节点为空。
- 3、⼆叉树节点数最⼤不超过100。

**注**:输⼊字符串格式是正确的，⽆需考虑格式错误的情况

**输出描述**

- 输出⼀个字符串，为⼆叉树中序遍历各节点值的拼接结果。

**⽰例：**

```
输⼊：a{b{d, e{g,h{,I}}},c{f}｝
输出：dbgehiafc
```
