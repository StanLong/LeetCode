# 最大时间

**题目描述**

给定一个数组，里面有 6 个整数，求这个数组能够表示的最大 24 进制的时间是多少，输出这个时间，无法表示输出 invalid。

**输入描述**

输入为一个整数数组，数组内有六个整数。

输入整数数组长度为 6，不需要考虑其它长度，元素值为 0 或者正整数，6 个数字每个数字只能使用一次。

**输出描述**

输出为一个 24 进制格式的时间，或者字符串”invalid“。

**备注：**

输出时间格式为 xxxxx 式。

**示例 1**  输入输出示例仅供调试，后台判题数据一般不包含示例

**输入**

```
[0,2,3,0,5,6]
```

**输出**

```
23:56:00
```
